### [242\. 有效的字母异位词](https://leetcode-cn.com/problems/valid-anagram/)

Difficulty: **简单**

给定两个字符串 _s_ 和 _t_ ，编写一个函数来判断 _t_ 是否是 _s_ 的字母异位词。

**示例  1:**

```
输入: s = "anagram", t = "nagaram"
输出: true
```

**示例 2:**

```
输入: s = "rat", t = "car"
输出: false
```

**说明:**  
你可以假设字符串只包含小写字母。

**进阶:**  
如果输入字符串包含 unicode 字符怎么办？你能否调整你的解法来应对这种情况？

#### Solution

```javascript
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var isAnagram = function (s, t) {
  if (s.length !== t.length) {
    return false;
  }
  const map = new Map();
  for (let i in s) {
    if (!map.has(s[i])) {
      map.set(s[i], 1);
    } else {
      map.set(s[i], map.get(s[i]) + 1);
    }
    if (!map.has(t[i])) {
      map.set(t[i], -1);
    } else {
      map.set(t[i], map.get(t[i]) - 1);
    }
  }
  for (let i of map) {
    if (i[1] !== 0) {
      return false;
    }
  }
  return true;
};
```
